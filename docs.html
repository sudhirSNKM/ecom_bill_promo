Welcome to MarsCart

Last updated March 4, 2026 · 5 min read

MarsCart is the all-in-one retail platform for modern boutiques. This documentation covers everything you need to set up your store, manage your team, and grow your business — from POS setup to API integration.

What is MarsCart?

MarsCart is a multi-tenant SaaS platform built specifically for modern retailers. It combines POS, inventory management, e-commerce, analytics, and branch management into a single unified system — accessible from any device.

💡
New to MarsCart? Start with the Quick Start guide to get your first boutique configured in under 30 minutes, including your first product and a live test transaction.

Core Concepts

Companies and Branches

The top-level entity in MarsCart is a Company. A company can have one or more Branches — physical store locations or warehouses. Each branch has its own inventory, sales data, and staff, while sharing the company's product catalog and customer base.

User Roles

Role Access Level Typical Use
Super Admin Full platform access Platform operations team
Company Admin All branches in their company Boutique owner / GM
Branch Admin Single branch only Store manager
Cashier POS + limited inventory Sales associate
Customer E-commerce storefront only End customer

Quick Setup

After signing up, follow these steps to get your boutique live:

// Step 1: Create your company profile
POST /api/v1/companies
{
  "name": "Maison Éclat",
  "country": "AE",
  "currency": "AED"
}

// Step 2: Add your first branch
POST /api/v1/branches
{
  "company_id": "cmp_xxx",
  "name": "Dubai Marina Location"
}

// Step 3: Add products via dashboard or CSV import
