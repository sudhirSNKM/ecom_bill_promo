API Reference

The Zenith REST API lets you build integrations, automate workflows, and access your retail data programmatically. All endpoints return JSON and require a Bearer token for authentication.

Base URLhttps://api.zenith.app/v1

Authentication

POST /auth/login Obtain an access token

Authenticate a user and receive a JWT access token and refresh token.

Request Body
emailreqstringThe user's email address.
passwordreqstringThe user's password (min 8 chars).
Response
access_tokenstringJWT valid for 1 hour.
refresh_tokenstringUsed to refresh the access token without re-login.
userobjectUser profile including id, name, role, and company.

Products

GET /products List all products

Returns a paginated list of products for the authenticated company.

Query Parameters
pageintegerPage number (default: 1).
limitintegerResults per page, max 100 (default: 20).
branch_idstringFilter by branch ID to get branch-specific stock levels.
searchstringSearch by product name, SKU, or barcode.
POST /products Create a product

Create a new product in the company's catalog.

Request Body
namereqstringProduct display name.
skureqstringUnique stock keeping unit.
pricereqnumberSelling price in the company's base currency.
categorystringCategory name or ID.
variantsarrayArray of size/color variant objects.
imagesarrayArray of image URLs.
